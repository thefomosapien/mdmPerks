Thanks for buying Booker

The Booker template is a simple HTML5 theme specially designed for showcasing your eBook in quality and custom style. Booker is a single landing page dedicated to promoting and selling your ebook. Booker is not only a responsive theme meaning it is optimised for multiple device sizes, Booker is also retina ready, meaning no more blurry images on your HiDPI and retina displays. Remember Retina or HiDPI requires images twice the size of normal images.

How to use Booker?

Booker was built for non-web designer and developers, and in a few moments you can customise Booker to suite your own app and brand.

Important Note:
Remember this theme is HiDPI, so all your images need to be delivered in Hi Res to avoid looking blurry on HiDPI and Retina displays. It is always best to work from the supplied template PSD which is Retina Sized.

Changing Logo's

Booker pulls it's logo's and all other images from the img folder. Simply replace the logo files with your own, it's easiest to just overwrite the existing logo files with your own logo. Remember to include HiDPI versions.

Because Booker is Retina ready, and Retina Images are downscaled for non-HiDPI or Retina screens, you may have to update the size of the allocated area for your logo to avoid distortion or cropping. Open the main.css file and look for 

 /* ===================================================

    LOGO

    ================================================ */
Here you can see the width properties. Match the width to suite the size of your logo when halved or divided by 2. So for example, you will see that the provided logo is actually 446px wide, however we half the size in our CSS to 223px wide to accomodate for Standard and Retina Screens simultaneously.

Changing Call to Action text

Booker comes with a default area for Call to Action text or Headlines. A main heading, sub heading and feature button are provided. To update these look in the HTML for; 

 <- Start Call To Action ->

And replace the existing headline with your own compelling text. Remember to update the href="#Your Link Here" with your actual link to the downloadable eBook.

Changing Book Covers

Changing Book Cover Images

The book cover artwork can be simply updated in the supplied PSD file. Open it and go to the Head Section and then Books layer group. Here you will find 5 layers labeled 1 Book Cover, 1 Book Back, 2 Book Cover and so on. These layers are smart object layers and are turned off by default. Double click on the desired Book Cover layer, this activates the smart object in a new window. In the smart object window, paste your own cover image making sure to fill the complete area, save and close this window, and the Book cover will be updated.

Note:
the Book Back layers are shape layers with no artwork included. They are only intended to be colored layers to match the Cover color. To update these, double click on the Book Back layer to open in the smart object window, then double click again to bring up the color changer, now change to match the color of the front cover, save and close.

Note:
Being optimised for all devices, this theme uses 2 separate book images, one for wide displays and one for handheld displays. See the change when you resize your browser window. Be sure to update both versions of these images.

Updating Sample Thumbnails

Booker come with a responsive lightbox to showcase pages from your eBook. There is a default list of 6 pages you can highlight.

To change these images to your own, look in the HTML for

<- Start Page Thumb ->

This whole block of code is required for the thumbnail, the hover effect and the lightbox popup. In the relative block, simply update the "img/image.jpg" path to the path/name of your own image and do the same for the thumbnail image also "img/thumb.jpg"

To add or delete additional thumbnials, simple add or delete the required "Start Page Thumb" code block.

Changing Reviews

In the HTML look for the comments labelled 

<- Quote  ->
and change the filler text to your own awesome reviews/ testimonials.

Reviews also have an Author or by area, simply update the h6 tag for each review/Quote

Changing Menu Links

Booker has three menu areas, in the header area this is duplicated on all 3 main sections. In the Header area, to update the Download menu, look in the HTML file, the download menu item in the header and replace "#Your Link Here" with your own URL/link.

As menu links are unique to each menu section you will have to update this across all 3 menu sections in your HTML.

Email Sign Up

In your HTML file look for:

<- EMAIL NOTIFY SECTION ->

Simply copy in your email sign code, this should be provided by your email marketing provided, (mailchimp, campaignmonitor or similar), And copy the class names from the existing form field and button, onto the newly pasted in field and button.

Changing CSS

Whilst Booker is pretty kick ass already you may want to get advanced and change the colours of backgrounds, links, text etc. You can do all of this by editing the main.css file. The main.css file is well commented to highlight to most common changeable aspects like link colours etc.

Google Analytics

At the very bottom of the HTML you will find a Google Analytics code snippet to use. Simply replace the UA ID number with your own to set up Google Analytics and start receiving stats and figures about your webpage, or delete this whole section and copy and paste in the code snippet Google provides.

Also DO NOT FORGET to update the title and description of your web page. You can find these at the very top of the HTML file, just look for

<title>Booker</title>
 <meta name="description" content="">
and update them to you own apps name and a description between the content="" for search engines.
